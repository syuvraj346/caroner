# CarOner API

Planned NestJS backend for CarOner phase 1.

## Primary modules
- auth
- users
- vehicles
- vendors
- service catalog
- bookings
- reviews
- notifications
- support
- admin

## Planned cross-cutting capabilities
- JWT authentication
- OTP flow
- role-based authorization
- request validation
- OpenAPI documentation
- audit logging
- background jobs for reminders and notifications

## Suggested initial setup
- NestJS
- TypeScript
- Prisma or TypeORM
- PostgreSQL
- Redis integration
- Swagger
