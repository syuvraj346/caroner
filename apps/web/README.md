# CarOner Web App

Planned role-aware Next.js application for:
- customer web experience
- vendor portal
- admin portal

## Scope
Phase 1 focuses on a single web codebase with route separation by role.

Suggested route zones:
- `/` public landing and marketing pages
- `/app` customer application
- `/vendor` vendor portal
- `/admin` admin portal

## Planned concerns
- authentication and session management
- service discovery and booking flow
- profile and vehicle management
- vendor service management
- admin operations dashboard

## Suggested initial setup
- Next.js
- TypeScript
- Tailwind CSS
- React Query
- form validation library
- role-based layout shell
